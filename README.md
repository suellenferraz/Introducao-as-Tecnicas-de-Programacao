# 💻 Introdução às Técnicas de Programação

Este repositório foi criado para ajudar você a entender os exercícios da disciplina de **Introdução às Técnicas de Programação - IMD/UFRN**. Além disso, você pode utilizá-lo como uma ferramenta adicional para aprofundar seus estudos em programação. Sinta-se à vontade para explorar e aplicar os conceitos abordados!

#

Como complemento, este repositório inclui exercícios de duas plataformas: **Beecrowd** e **YouCoding**, cobrindo diversos tópicos como condicionais, estruturas de repetição, funções e muito mais.
